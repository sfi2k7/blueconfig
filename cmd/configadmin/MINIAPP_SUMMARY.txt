          ├─ DefaultProperties (default)
          └─ TimeseriesDashboard (__type: "timeseries")
              ├─ SensorList
              ├─ SensorControls
              ├─ SensorChart
              ├─ EventLog
              └─ SensorDetail

How It Works:
-------------
1. Click node in tree
2. RightPanel loads node properties
3. Check __type property
4. MiniAppContainer loads appropriate component:
   - __type: "timeseries" → TimeseriesDashboard
   - __type: undefined → DefaultProperties
   - __type: unknown → DefaultProperties (fallback)
